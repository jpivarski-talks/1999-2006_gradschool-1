BuildGeantGeometry - Manages Geant Geometry for Cleog under Suez

Source Code

See BuildGeantGeometry.h and BuildGeantGeometry.cc

Description

  • Responsibilities
    • Manage a STL map of different geometry streams with their synch values
    • Creates Geant geometry
    • Check that the geometry is consistent. Look at synch values of the different geometry stream and see if it's changed.

  • Geant Geometry Creation
    • Extract detector geometry from the Frame
    • Extract G3iExtraVolSpecsStore for Geant specific geometry info
    • Create Geant volumes via Brian's G3iCreatePlacedVolumes routine

  • Status
    • Currently extracts SV, DR, CC detector geometry (with perfect alignment)
    • Extracts CC G3iExtraVolSpecsStore - only available for the CC
    • Does not create geometry via old Fortran geometry libraries

  • TODO
    • Use aligned detector geometry stores when available
    • Create map of synch values with the different geometry streams
    • Check that the geometry is valid, else abort job
