Cleog and Geant calls by the MCResponseProd Producer

  • setup_geant_()
    called in the MCResponseProd constructor.

  • set_geant_run_()
    called by MCResponseProd::updateCleogBeginRun - tells Geant what the run number is which is set by the Suez system.

  • setup_kuip_()
    called by MCResponseProd::updateCleogBeginRun after the geometry is created if the user has set the Suez run time parameter UseKuip true. The KUIP interface is available to Geant/Cleog "experts". Standard running of cleog will not require KUIP and most users will only use the Suez interface.

  • cleog_update_br_()
    Cleog update begin run constants.

  • usbrun_()
    Geant user begin run routine.

  • cleog_trig_()
    Routine that triggers Geant for the event. This replaces cgtrig to avoid a name conflict.

  • uglast_()
    Geant user end of job call, called in the MCResponseProd destructor.

  • paexit_()
    CERN routine to exit & cleanup from paw, called in the MCResponseProd destructor.
