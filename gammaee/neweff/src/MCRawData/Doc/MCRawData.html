MCRawData

This document explains the functioning of the MCRawData library.

This library contains the classes which define and fill the raw data objects from the MCResponse of each sub-detector. The library has a producer which holds proxies for all the RawData objects. The faultHandler() for each proxy, in turn, asks for the information it needs to assemble the raw data, probably an FATable< MCXXXResponse >, to which it adds noise, digitizes, sparsifies, etc. Right now they are mostly dummy skeletons which do little except set the structure for returning the information.
